My Coding Journey

08/04/2023

Correlation One is actually the start of my coding career! Difficult to say as it's only the beginning of the road still, but I'm currently enjoying the course immensely. 08/11/2023

Throughout my time so far in Correlation One, I feel as though I have learned a lot and applied those skills well into my projects. I hope to learn more new tools I can use in my future work, as well as meet more intruiging people along the way.

08/18/2023

Things have started picking up, but currently still enjoying the course.
